My Work

Your notepad in one app

Skills: Sketch, Craft, Unsplash

Designed a functional note taking app within a time limit. This assignment was from the Careerfoundry, and they asked to create five screens for a note taking app. This task did not ask for UX deliverables too.

  • To design a functional app, I made the app into an easy navigation flow with its simplified functions. Added an Upload File, Pencil, and Shared buttons.
  • Easy to use scheduling page, with its swipe left/right calendar. Users have the option to include their notes, files and or lists, onto the Event page.
  • The add button opens up the navigation bar, which gives the user option to switch to other boards.
